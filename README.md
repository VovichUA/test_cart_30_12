```Як запустити проект?```

```1. composer install```

```2. npm install```

```3. cp .env.example .env```

```4. ./vendor/bin/sail up -d```

```5. ./vendor/bin/sail artisan key:generate```

```6. ./vendor/bin/sail artisan migrate```

```7. ./vendor/bin/sail artisan db:seed --class=ProductSeeder```

```8. npm run dev```

переходимо на http://localhost:8080/
